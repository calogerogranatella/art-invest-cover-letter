# Art-Invest Bewerbungsschreiben

Eine digitale, interaktive Bewerbung, die speziell für die Rolle als **Fundmanager Hotelimmobilien** bei Art-Invest entwickelt wurde.

Anstelle eines klassischen, statischen PDFs nutzt dieses Projekt eine maßgeschneiderte "Bento Box"-Benutzeroberfläche, um mein strategisches Mindset, meinen operativen Hintergrund und meine technischen Fähigkeiten in einem modernen und ansprechenden Format zu präsentieren. 

🔗 **[Zur Live-Anwendung](https://calogerogranatella.github.io/art-invest-cover-letter/)**

## Wie es entwickelt wurde
Diese Anwendung wurde als leichtgewichtige Single-Page-Web-App (SPA) entwickelt, um schnelle Ladezeiten und nahtlose Kompatibilität über alle Endgeräte hinweg zu gewährleisten.
* **Struktur & Styling:** Erstellt mit HTML5 und CSS3. Nutzt Tailwind CSS für effizientes Styling und natives CSS Grid, um das komplexe, responsive "Puzzle"-Layout zu ermöglichen.
* **Interaktivität:** Vanilla JavaScript steuert die responsiven Verhaltensweisen, einschließlich eines maßgeschneiderten Akkordeon-Effekts für Touchscreens auf Smartphones und Tablets.
* **Design:** Maßgeschneiderte, monochromatische grüne Farbpalette, die eine institutionelle, aber dennoch unternehmerische Identität (die "Day One" Kultur) widerspiegelt.
